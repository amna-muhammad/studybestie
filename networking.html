1. DNS Resolution

DNS converts studybestie.live into server IP addresses so browsers can find the site.

nslookup studybestie.live

Name:    studybestie.live
Addresses:  18.208.88.157
            98.84.224.111

Result: DNS resolves correctly to two public IPv4 addresses (load-balanced edge hosting).

2. IP Addressing

Current observed addressing for this domain is IPv4:

  • 18.208.88.157
  • 98.84.224.111

These are the destination addresses clients connect to after DNS lookup.

3. Protocols (HTTPS + Status Codes)

StudyBestie is served over HTTPS, which encrypts traffic with TLS.

curl -I https://studybestie.live
HTTP/1.1 200 OK
Server: Netlify
Content-Type: text/html; charset=UTF-8
Strict-Transport-Security: max-age=31536000

Status 200 OK confirms the main page request succeeded.

4. Request and Response Headers

Headers prove the protocol details and server behavior.

Request headers (example)
GET / HTTP/1.1
Host: studybestie.live
User-Agent: curl/8.18.0
Accept: */*

Response headers (example)
HTTP/1.1 200 OK
Content-Type: text/html; charset=UTF-8
Cache-Control: public,max-age=0,must-revalidate
Server: Netlify
Strict-Transport-Security: max-age=31536000

5. How TCP/IP Fits

  • DNS resolves studybestie.live to IP addresses.
  • The browser opens an HTTPS connection over TCP/IP to one of those IPs.
  • The server returns HTML/CSS/JS with status code 200 and response headers.
  • The browser renders the StudyBestie app to the user.
